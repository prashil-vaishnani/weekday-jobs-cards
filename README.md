
# Weekday Jobs Search


## Installation

clone the repo
```bash
git clone https://github.com/prashil-vaishnani/weekday-jobs-cards.git

```

go to folder and Install dependancy with npm

```bash
  cd weekday-jobs-cards
  npm install
```

then run locally
```bash
  npm run dev
```
